## 系統說明

> Node.js version 16.10.0

> 運行專案

```
npm install
npm run dev
```

> 打包專案

```
npm run build
```

## 資料夾說明

> /src/components 依 router 頁面存放各組件<br/>
> /src/composable gsap 動畫組件
